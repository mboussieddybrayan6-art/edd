# Calculatrice Pro – Description du projet

Créer un site web moderne, rapide et entièrement responsive nommé **Calculatrice Pro**.

La page d'accueil doit présenter un design élégant avec un fond sobre, un logo de calculatrice et un titre bien visible. Sous le titre, afficher une courte présentation expliquant que cette plateforme permet d'effectuer des calculs précis, rapides et sécurisés.

La calculatrice doit offrir les fonctionnalités suivantes :

* Calculs de base : addition, soustraction, multiplication et division.
* Pourcentages.
* Racines carrées.
* Puissances et exposants.
* Parenthèses pour les calculs complexes.
* Changement de signe (+/-).
* Historique des calculs.
* Boutons **Effacer (C)** et **Supprimer (⌫)**.
* Copie du résultat en un clic.
* Prise en charge des décimales.
* Affichage clair des opérations et du résultat.

Le site doit inclure :

* Un mode clair et un mode sombre.
* Des animations fluides sur les boutons.
* Une interface professionnelle inspirée des applications modernes.
* Une utilisation parfaite sur ordinateur, tablette et smartphone.
* Une excellente rapidité de chargement.

Ajouter un menu de navigation comprenant :

* Accueil
* Calculatrice
* Historique
* À propos
* Contact

Créer également une section « À propos » expliquant que **Calculatrice Pro** est un outil conçu pour offrir des calculs fiables, rapides et précis, destiné aux étudiants, professionnels, commerçants, ingénieurs et à toute personne ayant besoin d'une calculatrice performante.

Prévoir une page « Contact » avec un formulaire comprenant le nom, l'adresse e-mail, le sujet et le message.

Le site doit être optimisé pour le référencement (SEO), sécurisé, accessible et respecter les bonnes pratiques du développement web afin d'offrir une expérience utilisateur de qualité.
